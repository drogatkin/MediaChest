Your database requires upgrade 

The MediaChest starting from build 60 uses a slightly different database schema than R-PhotoOrganizer used. Therefore, you need to upgrade your your database schema to continue use album feature of the MediaChest. 

Recommended upgrade procedure

Open options window and select album panel.

If album option Maintain Album Folders is set off

  1. Uncheck this option
  2. Press button Drop tables and confirm the operation
  3. Select menu File/Import/From disk... and select directory specified in album options as Maintain Album Folders
  4. Set Maintain Album Folders off again

Notice that if you had the same picture shared by multiple albums, then you will need manually assign the picture to some albums.

If album option Maintain Album Folders is not checked off

  1. Select menu File/Export/To disk.. and select a directory for export where enough free space for entire your photo collection
  2. Press button Drop tables and confirm the operation
  3. Select menu File/Import/From disk... and select directory specified in step 1

Notice that old picture files location will not be used anymore.

As only CSV import/export will be ready, the procedure of migration to new database schema will be less painful.
